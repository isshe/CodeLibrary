## 4. 使用场景是什么？
* 当需要把待执行的动作抽象出来，参数化给某对象时。
* 在不同时刻指定、排列和执行请求。
* 支持取消/撤销操作。
    * 将操作前的状态保存起来，取消操作时，用此状态来消除操作的影响。
    * 进行逆操作应该也是一种方法。
* 支持操作日志（修改日志）。（提供加/装载和存储操作）
    * 当系统奔溃时，可以根据日志重新将操作做一遍，来恢复奔溃前的状态。
* 用构建在原语操作上的高层操作构造一个系统。【？！】
