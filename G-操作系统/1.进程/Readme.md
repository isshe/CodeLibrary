# 进程
进程是操作系统中最核心的概念。
* 定义：一个执行中的程序的实例。（对正在运行程序的一个抽象）
* 是资源分配的基本单位。


## 1. 进程创建与终止
* 导致`进程创建`的主要事件：
    * 1）系统初始化；
    * 2）正在运行的程序执行了创建进程的系统调用；
    * 3）用户请求创建一个新进程；
    * 4）一个批处理作业的初始化；
* 导致`进程终止`的主要事件：
    * 1）正常退出（自愿的）；
    * 2）出错退出（自愿的）；
    * 3）严重错误（非自愿）；
    * 4）被其他进程杀死（非自愿）；

### 2.2 基本状态
* 1）运行态：进程实际占用CPU；
* 2）就绪态：可运行；
* 3）阻塞态：除非某种外部事件发生(使进程满足运行条件)，否则进程不能运行。

### 2.3 引入的状态：

## 3. 进程实现
