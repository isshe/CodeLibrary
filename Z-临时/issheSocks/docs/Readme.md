
# 总体结构
```
                           (B)----------------------------------------------------+
    (A)-----------+        | +--------------------+       +---------------------+ |
    | User Client |  <---> | | isshe_proxy server | <---> | isshe tunnel client | |
    +-------------+        | +--------------------+       +---------------------+ |
                           +----------------------------------------↑-------------+
                                                                    |
                                                                    |
                           (C)--------------------------------------↓-------------+
  (D)-------------+        | +--------------------+       +---------------------+ |
  | Public Server |  <---> | | isshe_proxy client | <---> | isshe tunnel server | |
  +---------------+        | +--------------------+       +---------------------+ |
                           +------------------------------------------------------+
```
* A: 用户主机上的socks5客户端。通过socks5连接到服务器
* B：用户主机上的代理服务器+隧道客户端。
* C：外部主机上的隧道服务器+代理客户端(代理用户的请求)。
* D：外部的公共服务器
